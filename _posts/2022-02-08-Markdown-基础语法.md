# **Markdown 基础语法**

## Markdown 标题语法

{% highlight js %}
# Heading level 1
## Heading level 2
//...
{% endhighlight %}

还可以在文本下方添加任意数量的 == 号来标识一级标题，或者 -- 号来标识二级标题。

{% highlight js %}
Heading level 1
===============

Heading level 2
---------------
{% endhighlight %}
==注意 #F44336== #与标题有空格
